# Inputs

This folder contains the canonical bundled assets used by the ReGlyco Colab demo.

- `REPO.pdb`: default recombinant EPO target used by the notebook quick demo.
- `REPO_complex.pdb`: bundled example complex used to derive the deterministic preset patch for the quick demo.

The notebook uses `REPO.pdb` numbering directly for the bundled target, including the O-glycosylation site at `A126`.
